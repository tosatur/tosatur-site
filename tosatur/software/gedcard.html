GEDCARD

Gedcard is a Python script that converts a GEDCOM (.ged) file to a list of contacts in a vCard (.vcf) file.

It took me about an hour to write.

There are flags for including the gender in the vCard, and including deceased people.

I will (not) expand this to include support for JSContact and xCard.
