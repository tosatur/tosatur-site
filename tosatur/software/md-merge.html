.md Merge

.md Merge is an Obsidian plugin I wrote, which takes all the Markdown files in a given folder, and merges them into a single file.

I wrote this plugin in a single afternoon. It is pretty configurable, and runs quickly on large vaults. You can use it via a button on the ribbon, or via the command palette.

It is currently awaiting review by the Obsidian team, to be added to the official list of community plugins.
