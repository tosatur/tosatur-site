CHAIRPUSHER

Chairpusher is the earliest published "game" i've made. It is a "game" in which you (can) push chairs off a platform.

Fueled by frustration from failing to finish a game project, I channeled my rage into creating the most pointless, low-effort game imaginable. Thus, Chairpusher was born.

All you can do is pick up chairs, push them around, and throw them off the platform. once any object (including the player) falls around 100m below the platform, a toilet flushing sound is played, and the game crashes. There is also a floating Chowder bean sphere.

The filesize is a whopping 380mb. Sorry about that; I accidentally left in the default Unreal Engine starter content.
